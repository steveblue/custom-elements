Custom Elements

Autonomous custom elements. These components extend from HTMLElement, can attach ShadowDOM. To make autonomous custom elements accessible, you must provide custom logic. Autonomous custom elements are out of the document flow. You can make autonomous custom elements accessible using `tabindex` and listening KeyboardEvent consistent with other elements in the flow.

Use tab to navigate to the MyListComponent. Then press the up and down arrow keys, followed by Enter to select an item in the MyListComponent.

Customized built-in elements. These components retain the default characteristics of the HTML Elements they extend from. If you extend from a component that already has behaviors consistent with accessibility, the component will retain those behaviors. Elements like HTMLButtonElement and HTMLInputElement are by default in the flow of the document. The user can tab to these elements by default without the use of tabindex.
